# Titel
Snappthis, dit is een webapplicatie waar je foto's op kan plaatsen voor gezamenlijke reflectie en creativiteit.

## Beschrijving
### Hoe werkt de webbapplicatie en wat kan je ermee?

Snappthis is een mobile-first webapplicatie waar verschillende gebruikers samen mindmaps maken. Binnen de mindmaps worden foto's met elkaar gedeeld voor nieuwe idee√´n, creativiteit en binding. 

De applicatie laat gebruikers anderen uitnodigen om mee te doen, zodat een groep verschillende perspectieven kan verzamelen rond een thema. Snappthis wordt vaak gebruikt door studenten, docenten en onderzoekers. 

Je kunt in de app foto's liken, disliken en bewaren in jouw favorieten. Op jouw eigen profiel vind je verder een overzicht van alle mindmaps waarin jij bent betrokken.
